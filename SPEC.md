# SPEC.md

## App
TaskRemind Pro

## Platform
Android only

## What It Does
Daily task reminder app with floating overlay popup
that appears over any screen even when app is closed.
Exactly like Truecaller incoming call screen.

## Features
- Create, edit, delete tasks
- Set daily reminder time per task
- Global on/off toggle
- Floating overlay when reminder fires:
  shows task title + Done / Snooze 10min / Dismiss
- Settings screen: dark mode toggle, permissions guide

## Constraints
- Android only, no iOS
- 100% offline, no internet
- No cloud, no Firebase, no backend

## Done Means
- APK installs on Android phone
- Task creation works
- Reminder fires at set time
- Floating overlay appears automatically
- Done and Snooze buttons work
